SUMMARY

This is an interactive HTML INF viewer for the AC97 WDM Audio Driver Sample INF only. Use this tool to learn more about the different sections in the AC97 Audio Driver Sample INF file.  Open the INFViewer.html page, to the left side you will see the AC97SMPL.INF file and clicking on any of the highlighted links will display a brief explanation of it on the right side.
