Server Availability Virtual Drivers

When used with Failover BIOS, Server availability can be improved using the following drivers available with Windows Server 2003:

·        Watchdog Timer

·        Non-Volatile RAM (NVRAM)

·        Local Display

·        Local Keypad

The Watchdog Timer automatically reboots the server if the server fails to respond in a predefined time period. The NVRAM stores a count of the number of times a system tries to unsuccessfully boot. The Failover BIOS automatically switches the boot partition after a predefined number of attempts. These features combine to provide the ability to boot an appliance automatically from a second hard disk if the first hard disk fails.

The Local Display and Local Keypad allow the OEM to build a liquid-crystal display (LCD) and 6-key keypad that can be used to set an IP address or display status messages.

Additional information can be found in the Windows Server 2003 Device Driver Kit (DDK).

Installation

To install the three virtual drivers, use the DevCon utility (Devcon.exe).  DevCon is a command-line utility that provides an alternative to Device Manager. It allows you to enable, disable, restart, update, remove, and query individual devices or groups of devices. DevCon provides information that is relevant to the developer and is not available in Device Manager.  You can download DevCon from the Microsoft support website; see Knowledge Base (KB) article number Q311272,

The source code and documentation for DevCon are provided in the Windows XP DDK (which is available from http://www.microsoft.com/ddk/ ) in %winddk%\Src\Setup\Devcon.

To install the drivers, copy all the INF and binary files from the binaries directory to a temporary directory on the target system, and then at a command prompt, type the appropriate command:

·        To install the NVRAM driver type the following command:
DEVCON INSTALL MSNVRAM.INF MSNVRAM

·        To install the Local Display driver, type the following command:
DEVCON INSTALL MSDISP.INF MSDISP

·        To install the Local Keypad driver, type the following command:
DEVCON INSTALL MSKEYPAD.INF MSKEYPAD

Execution

Once the drivers are installed and loaded, the system is ready to exercise the driver stack. The SAGUI application provides a local display, and keypad input.
