WacomPen HID Minidriver

[This is preliminary documentation and subject to change.]

SUMMARY

This directory contains code that is specific to the sample WacomPen HID minidriver for a Tablet PC. The WacomPen driver operates pen devices from Wacom Technology Corporation that use a 16550 UART-compatible interface. For an overview of this driver and the other sample HID minidrivers for a Tablet PC, see src\wdm\hid\tablet\readme.htm.
