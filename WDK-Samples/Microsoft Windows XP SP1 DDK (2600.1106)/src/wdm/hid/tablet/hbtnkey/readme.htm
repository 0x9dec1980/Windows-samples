HBtnKey HID Minidriver

[This is preliminary documentation and subject to change.]

SUMMARY

This directory contains code that is specific to the sample HBtnKey HID minidriver for a Tablet PC. The HBtnKey driver operates a button device that uses an i8042 port-compatible interface. For an overview of this driver and the other sample HID minidrivers for a Tablet PC, see src\wdm\hid\tablet\readme.htm.
